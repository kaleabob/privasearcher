Introduction

The PrivaSearch API provides programmatic access to web, image, video, and news search. No authentication required for basic usage. Completely free and private.

Base URL: https://privasearch.render.com/api

Available Endpoints

  • GET /search - Web search
  • GET /images - Image search
  • GET /videos - Video search
  • GET /news - News search
  • GET /health - Status check

Web Search

GET /api/search

Search the web across multiple search engines.

Query Parameters

ParameterTypeRequiredDescription
qstringYesSearch query
enginestringNoduckduckgo, bing, or google (default: duckduckgo)
ssstringNoSafe search: off, moderate, strict (default: off)
skstringNoSerper API key (for Google results)

Response

{ "organic_results": [ { "title": "Result title", "link": "https://example.com", "snippet": "Result description..." } ], "provider": "DuckDuckGo" }

Example

GET /api/search?q=python+programming&engine=duckduckgo

Image Search

GET /api/images

Search for images with thumbnail URLs.

Query Parameters

ParameterTypeRequiredDescription
qstringYesSearch query
ssstringNoSafe search: off, moderate, strict (default: off)

Response

{ "organic_results": [ { "title": "Image title", "link": "https://example.com/image", "imageUrl": "https://thumbnail.url" } ], "provider": "Images (DDGS)" }

Video Search

GET /api/videos

Search for videos with metadata (duration, channel, thumbnail).

Query Parameters

ParameterTypeDescription
qstringSearch query (required)

Health Check

GET /health

Check if the API is online and responding.

Response

{ "status": "ok", "timestamp": 1781922529.625 }

Usage Examples

JavaScript / Fetch

const query = 'machine learning'; const response = await fetch( `https://privasearch.render.com/api/search?q=${encodeURIComponent(query)}` ); const data = await response.json(); console.log(data.organic_results);

Python / Requests

import requests query = 'climate change' response = requests.get( 'https://privasearch.render.com/api/search', params={'q': query, 'ss': 'moderate'} ) data = response.json() print(data['organic_results'])

cURL

curl "https://privasearch.render.com/api/search?q=artificial+intelligence"

Authentication

No API key required! PrivaSearch API is completely open and free. We don't track API usage.

Optional: Use sk parameter with your Serper API key to access Google results.

Rate Limiting

No official rate limits. Please use responsibly. Excessive requests may be throttled.
